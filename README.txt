Projeto Amigo Secreto
Este projeto é uma aplicação web simples para organizar um sorteio de amigo secreto. Através dela, você pode adicionar nomes dos participantes, realizar o sorteio e exibir os resultados de forma prática e visual.

Funcionalidades
Adicionar nomes dos participantes pelo campo de texto.

Permite adicionar nomes pressionando o botão Adicionar ou a tecla Enter.

Listagem dinâmica dos participantes adicionados.

Sorteio automático do amigo secreto, garantindo que ninguém tire a si mesmo.

Resultado do sorteio exibido na tela.

Imagem do cabeçalho com efeito de fade que desaparece e reaparece a cada 5 segundos.

Design responsivo e visual moderno com cores vibrantes.

Como usar
Clone ou baixe este repositório.

Abra o arquivo index.html em seu navegador preferido.

Digite o nome dos participantes no campo de entrada e clique em Adicionar (ou pressione Enter).

Após adicionar todos os participantes, clique no botão Sortear amigo para ver o resultado do sorteio.

A lista de participantes e o resultado serão exibidos na página.

Estrutura dos arquivos
index.html — Estrutura da página web.

style.css — Estilos para o layout e visual da aplicação.

app.js — Código JavaScript para funcionalidades (adicionar, sortear, efeitos).

Tecnologias usadas
HTML5

CSS3

JavaScript (Vanilla)

Observações
Este projeto é inteiramente frontend, ou seja, roda no navegador.

Não envia emails automaticamente — para isso, é necessário um backend e servidor de email.

Pode ser facilmente ampliado para incluir funcionalidades extras.
